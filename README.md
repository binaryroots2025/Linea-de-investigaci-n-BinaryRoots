# Linea-de-investigaci-n-BinaryRoots
 “BinaryRoots” es un espacio académico y formativo creado con el propósito de fortalecer las competencias en lógica, algoritmia y resolución de problemas de los estudiantes de la Universidad de Pamplona, mediante la práctica constante y la participación en eventos de programación competitiva a nivel institucional, regional y nacional.
